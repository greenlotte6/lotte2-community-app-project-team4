## 핵심 경험 및 기술

**MSA 아키텍처 적용**

- API Gateway, 드라이브 서비스, 채팅 서비스, 통합 서비스를 분리하여 배포 및 운영
- 별도의 API Gateway를 통해 단일한 서비스 엔드포인트 구축
- Proxy Pattern을 적용하여 서비스 간 통신을 최소화
